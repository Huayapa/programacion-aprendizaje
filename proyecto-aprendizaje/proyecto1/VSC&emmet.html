ATAJOS DE VISUAL STUDIO CODE

Estos son los 19 atajos que a mi parecer son utiles:
- ctrl + (+ o -) para zoom del programa.
- ctrl + j o Ã± para la terminal y la consola de depuracion.

- alt + shift + a crear un comentario en html.

- ctrl + c copiar linea
- ctrl + v pegar linea.
- ctrl + x cortar linea.

ATAJOS EMMET

recuerda que estos atajos emmet son para crear etiquetas facilmente.

  • html crea la estructura del html.
  • crear nodos hijos con > ejemplo: (nav>ul>li).
  • crear hermanos con + ejemplo: (div+article+section).
  • crear nodos superiores ^ ejemplo: (ul>li>p^li>blockquote).
  • crear un multiplicador con * ejemplo: (nav>ul>li*5>a).
  • ul>li*3>(p+span)*2 no es lo mismo que ul>li*3>p+span*2 por eso el uso de () para la jerarquia.
  • nav# para las id. usar #. esto es inportante
  • nav. para las clases. usar .
  • nav[ ]para crear un enlace
