  • El fondo

    Que se puede aplicar con el background-color.

  • El elemento

Sistema Hexadecimal

Se llama Hexadecimal porque esta basada en el sistema de numeración Hexadecimal, quiere decir que contiene 16 digitos (0123456789ABCDEF) y solo acepta 6 digitos sin contar el (#).

Colores HEX

Para poder mostrar los colores debes saber primero que es RGB (RedGreenBlue) siendo asi los primeros dos digitos seran para el red, los segundos dos digitos seran green y los ultimos dos seran para el blue (Al principio debes poner el #).

#ff6600 

                    {
                        color: #ffffff; }color blanco
                        color: #ff0000; }color rojo
                        color: #00ff00; }color verde
                        color: #0000ff; }color azul
                        color: #000000; }color negro
                    }

Opacidad

Tambien podemos añadirle la Opacidad, para esto debemos poner 2 digitos adelante de los 6 digitos, para que lo interprete como el porcentaje (%) de la opacidad (de 1% a 99%). 

                    {
                        color: #00000050;
                    }

Sistema RGB

RGB (RedGreenBlue) es un sistema que consiste representar varios colores a travez de combinar el rojo, verde y azul.

El sistema RGB acepta 8 bits por canal (los bits solo aceptan 0 y 1), su valor maximo es 255, el mas bajo es 0 y puede aceptar 9 digitos (255, 255, 255), para poder aplicarlo debemos hacer uso de la funcion rgb().

Porques solo acepta 255?

Como el sistema RGB acepta 8 bits por canal, debemos elevar 2 a la potencia 8 que es igual a 256 (Solo acepta 255 porque el 0 cuenta a la hora de pogramar).

rgb(255, 255, 255) 

                    {
                        color: rgb(255, 255, 255); }color blanco
                        color: rgb(255, 0, 0); }color rojo
                        color: rgb(0, 255, 0); }color verde
                        color: rgb(0, 0, 255); }color azul
                        color: rgb(0, 0, 0); }color negro
                    }

Opacidad

Si quieres agregar opacidad debemos usar la función rgba() y debemos agregarlo en el 4 parametro.

Ten en cuenta que:

El numero 0 representaria el 0% de opacidad y el 1 el 100%, si quieres agregar una opacidad de 50% deberas usar los numeros decimales (0.50). 

                {
                  color: rgba(0, 0, 0, 0.50);
                }

Sistema HSL

HSL o HueSaturationLightness (Tono de Saturacion y Luminosidad) y acepta distintos valores.

Valores HSL

Debemos usar la función hsl() para poner los valores HSL.

  • El primero es hue (circulo cromatico) que acepta hasta 360°

    1. 0° } red

    2. 60° } yellow

    3. 120° } green

    4. 180° } cyan

    5. 240° } blue

    6. 300° } magenta

  • El segundo es Saturation (Intensidad de color) y se maneja por porcentajes.

    1. 0% } Escala de grises

    2. 100% } color puró

  • El tercer valor es Lightness (Luminosidad de color) y se maneja con los porcentajes.

    1. 0% } negro

    2. 50% } color puro

    3. 100% } blanco

              {
                background-color: hsl(60, 0%, 50%);
              }

Opacidad

Si quieres agregar opacidad debemos usar la función hsla() y acepta decimales igua a rgba(). 

              {
                background-color: hsla(60, 0%, 50%, 0.50);
              }

Unidades de medida en CSS

Las unidades de medida se puede clasificar en 2:

  1. Absolutas

    En Absolutas su valor no puede cambiar ya que son unidades reales.

    • cm, mm, pc, in, Q } Se usan para documentos con medidad para imprimir

    • pt (1/72 in) } puntos

    • px (1/96 in) } pixeles

  2. Relativas

    Su valor es relativo al contexto de cada unidad de medida.

    • em, rem, ex, ch } es al tamaño de la fuente

    • % } es al tamaño del contenedor

    • vw, vh, vmax, vmin } es al tamaño del viewport

recomendable

Recomendable usar las unidades de medida Relativa y visitar www.pxtoem.com te puede ayudar en las medidas.

recomendable

Antes de seguir debes saber a que en el alfabeto la letra "m" es la mas larga y la letra "X" es la mas alta de el alfabeto.

EMs (medida relativa)

Esta basada en la "m" de la fuente del elemento, me refiero a que los em siempren tomaran el tamaño base del elemento padre.

Por defecto el tamaño base es 16px asi que 1em = 16px pero si al elemento padre cambias su tamaño base a 24px, el 1em pasara a ser 24px.

Codigo

              }
              background-color: rebeccapurple;
              font-size: 2em;
              }

Ejemplo

soy un parrafo con 2em y mi elemento padre tiene un tamaño base de 16px eso quiere decir que mi tamaño sera de 24px

REMs (Medida Relativa)

La unidad de medida REMs o (Root em) esta basada en la altura "m" de la fuente del elemento raíz(html), esto hace que los rem sea mas facil de comprender que los em.

Es casi lo mismo que los Em, la diferencia es que solo estara haciendo caso a el elemento raiz (la etiqueta html), por defecto 1em = 16px y no se cambiar aunque el elemento padre tenga 24px.

Codigo

              }
                background-color: rebeccapurple;
                font-size: 1em;
              }

Ejemplo

Soy un parrafo con 1em y solo hago caso al elemento raíz.

EXs (Medida Relativa)

Los ex estan basado en la altura "X" de la fuente del elemento, tiene un parecido a los em la diferencia es que esta es sobre la altura y no la anchura.

Su valor de 1ex se puede proximar por 0.5em, Como se trata una de medida relativa es necesario un calculo matematico para determinar la medida real.

Codigo

              .rems {
                background-color: cadetblue;
                font-size: 24px;
                padding: 1ex 0.5ex;
                border: 5px solid black;
              }
            
              .rems-child {
                background-color: salmon;
                font-size: 2ex;
                padding: 1ex;
                margin: 0.5ex;
                border: 5px solid black;
              }

Los Porcentajes (%)

Los porcentajes (%) son relativos al tamaño del contenedor, esto quiere decir que su porcentaje siempre se definira al tamaño de su elemento. 

              {
                background-color: darkkhaki;
                width: 80%;
                font-size: 100%;
                padding: 2%;
                height: 100%;
              }

Debes saber que:

Si pones una medida a la altura (height) no lo aplicara, lo recomendable es que tu documento ocupe la altura dependiendo del elemento, pero si quieres ponerlo debes crear una caja con un tamaño definido para poder afectar al hijo.

Codigo

              height-parent {
                background-color: brown;
                width: 200px;
                height: 200px;
              }
            
              height-child {
                background-color: coral;
                width: 50%;
                height: 50%;
              }

Unidades del viewport

El viewport es relativo al tamaño del viewport, lo que quiero decir es que cada pantalla tiene distinto tamaño y el 100vh y 100vw representa el tamaño completo del viewport y existes 4 tipos de unidades de medida.

Unidades de medida del viewport:

  • vw

    vw o viewportWidth es para el ancho del viewport y va desde 1 al 100, es decir si tiene 1 agarra una pequeña proporcion del tamaño de la pantalla.

  • vh

    vh o viewportHeight es para la altura del viewport y va desde 1 al 100, es decir si tiene 1 agarra una pequeña proporcion del tamaño de la pantalla.

  • vmax

    vmax o viewportmax tomara entre vh y vw el que tenga mayor valor.

  • vmin

    vmin o viewportmin tomara entre vh y vw el que tenga menor valor.

Esto es muy util y el mas usado ya que nos permite adaptar la web a distintos tipos de pantalla, tambien podemos crear un "hero image".

Recuerda que:

  • las etiquetas body tienen margenes por defecto.

  • las barras de scroll del navegador tambien son parte del viewport.

  • es recomendable poner al width con porcentajes(%) y al height con tamaño del viewport(vh).

  • el viewport siempre estara basadas en el tamaño de la pantalla aunque este en una caja no le hara caso a la caja.

Codigo:

                
                  .hero-img-ejm {
                    background-color: #bdbdbd;
                    width: 100%;
                    height: 100vh;
                    text-align: center;
                    font-size: 5vh;
                    display: flex;
                    flex-direction: column;
                    justify-content: center;
                    padding: 5vh 2vh;
                  }

Cuando uso cada unidad de medida?

  • Las Absolutas

    Si tienes que diseñar una pagina para imprimir ahi nos combiene usar las absolutas (en general)

  • Unidades Relativas

    1. Todo lo que tenga que ver con tipografia se recomienda usar los rem

    2. Para padding, margin medidas externas podemos usar rem pero si los quieres que los tamaños sean proporcionales es mejor usar em

    3. Si quieres delimitar cierto contenedor o elemento nos combiene usar los ch

    4. Los porcentajes nos sirven para el diseño resposivo y que lo usaremos mucho.

    5. Las unidades del viewport (vh,vw) nos sirve cuando queremos tomar como referencia el tamaño de la pantalla y lo usaremos mucho igual que los porcentajes

Variables y funciones en CSS

CSS no es un lenguaje de programación pero cada vez enpieza a parecer a los lenguages.

Custom property (variables CSS)

Custom property o una variable (un espacio recervado para almacenar un dato), las variables deben estar definidas dentro de un selector.

Como los creo?

Para poder crear estas variables debemos usar las -- luego poner el respectivo nombre que tendra y por ultimo poner el valor que aplicara esta función. 

              
                soy-una-variable {
                  --ejemplo-de-nombre: red
                }

Como los Invoco?

Para poder invocar esta variable debemos hacer uso de la funcion var() y poner el nombre dentro (el nombre de la variable debe estar en el mismo selector para poder invocarlo) 

              
                soy-una-variable {
                  --ejemplo-de-nombre: red
                  color: var(--ejemplo-de-nombre);
                }

Recuerda

  1. Las variables deben estar en el mismo selector para funcionar.

  2. las Variables pueden tener el mismo nombre siempre y cuando esten en distintos selectores.

  3. Las variables solo afectaran a su selector y a sus hijos, no afectaran a los demas selectores que tengas en mismo nombre del valor.

Pero si quieres que una variable afecte a todos los elementos debemos crear nuestras variables en la etiqueta html, tambien podemos usar la seudoclase :root que es mejor porque tiene mayor especifidad.

Codigo:

                
                  .custom-props-1 {
                    --primary-color: gray
                    --font-size: 32px;
                    background-color: var(--primary-color);
                  }
                
              
                
                  .custom-props-3 {
                    font-size: var(--font-size);
                  }
                
              
                
                  .title-props {
                    --primary-color: navy
                    color: var(--primary-color, orange); }si la variable 
                                                      no funciona
                                                      ejecutara el
                                                      otro color
                  }

Funcion calc()

La función calc() nos sirve para hacer calculos aritmeticos.

Como lo creo?

Para poder crear esta funcion debemos hacer uso de la función calc() 

                
                  {
                    width: calc( 1px + 1px * 2px);
                  }

Recuerda que

  1. Recuerda poner los espacios en blanco, no todo junto ya que puedo provocar algunos problemas

  2. Lo mejor que puedes hacer es usar el calc con valores, porque si quieres hacer un cambio de tamaño en tu web demoraras menos tiempo.

Codigo

              
                .ch-10 {
                  --padding-size: 2ch
                  --num-ch: 10ch;
                  font-size: 2rem;
                  background-color: var(--default-bg-color);
                  width: calc(var(--num-ch) + var(--padding-size) *2);
                  padding: var(--padding-size);
                }

Función min() & max()

min() y max() hace que los valores de un selector se comporten dinamicamente y su funcion es que tomara el valor minimo o maximo de sus expresiones separadas por comas, dependiendo si es max o min

Codigo

              
                .ch-10 {
                  background-color: var(--default-bg-color);
                  margin-top: 1rem;
                  width: min(300px, 20vw, 20rem);
                  height: max(200px, 25vh);
                }

Funcion clamp()

clamp() es una función que puede recibir 3 valores (el primero el tamaño minimo, el segundo el tamaño ideal y el tercero el tamaño maximo) es un buen complemento para min() y max()

Debes saber que:

Para que funcione esta funcion debe tener unidades relativas y unidades absolutas al mismo tiempo.

Codigo

              
                .clamp-ejm {
                  font-size: clamp(2rem, 1rem + 3vw, 3rem);
                }

Fuentes externas

Las fuentes externas son sobre si quieremo agregar algun tipo de tipografia que no se encuentren por defecto.

Como lo creo?

  1. Para esto primero debemos usar una regla CSS que se llama @font-face

  2. necesitamos tener los archivos fisicos de la tipografia

  3. necesitamos que el formato de la tipografia sea woff o woff2 o ttf y Significa web open font format

Font-display

font-display es una propiedad de CSS que nos va decir como es que vas a cargar la fuente personalizada y tiene distintos valores:

  1. auto

    Permite que el navegador utilice el metodo predeterminado que suele ser block

  2. block

    oculta brevemente el texto hasta que la fuente haya sido descargado por completo.

  3. swap Recomendado

    Indica al navegador que utilice la fuente alternativa para mostrar el texto.

  4. fallback

    Es una mezcla de auto y swap

  5. optional

    Ocultara el texto, luego lo cargara con la fuente alternativa y finalmente aplicara la fuente personalizada

Codigo

                
                  @font-face {
                    font-family: "charlet"; }nombre clave
                    font-display: swap, sans-serif;
                    src: url("poner/direccion/tipografia.woff") format("woff");
                  }

                  .charset {
                    font-family: "charlet";
                  }

Google Font

Google fonts es una pagina que nos ayuda en encontrar tipografias y iconos para nuestra web, se puede agregar a tu web de 2 formas (por un link o por el @import)

Recomendaciones

  1. Cuando estes escojiendo algunas de las tipografias verifica bien si el contenido esta completo dependiendo de tus necesidades.

  2. Cuando estes agregando los estilos con diferentes grosores, solo agrega pocos ya que puede afectar el rendimiento de tu pagina.

  3. Usar 3 o mas tipografias es considerada una mala practica.

  4. Los links de la tipografia elegida deben estar antes que el link de CSS

  5. Para poder aplicarlo debes usar el codigo que esta mas abajo de donde copiastes el link.

Links para el head

              
                <link rel="preconnect" href="https://fonts.googleapis.com">
                <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
                <link href="https://fonts.googleapis.com/css2?family=Raleway:wght@100;400;800&display=swap" rel="stylesheet">

                {

Propiedad border-radius

Anteriormente ya vimos todas las propiedades del borde tambien aprendimos que hay una propiedad por cada uno de los bordes.

Pero nos olvidamos de la propiedad border-radius que nos permite redondear las cajas.

Border-radius

Border-radius es una propiedad shorthand eso quiere decir que son 4 propiedades en 1 y estas son sus 4 propiedades:

  1. border-top-left-radius

    Es la esquina superior izquierda

  2. border-top-right-radius

    La esquina superior de la derecha

  3. border-bottom-left-radius

    La esquina inferior derecha

  4. border-bottom-right-radius

    La esquina inferior izquierda

Extras

Tambien con el shorthand podemos añadir bordes diferentes solo debemos añadir mas medidas que afectaran de top, right, bottom, left.

Si queremos agregar bordes elipticos(un circulo ovalado vertical) tenemos que hacer uso de 2 valores en la definicion de border-radius y poner /en el medio de los 2.

Si quieres redondear una caja cuadrada debes usar 50%

Codigo

              
                .border-radius {
                  border-radius: 2rem;
                  border-top-left-radius: 4rem;
                  border-top-right-radius: 4rem;
                  border-bottom-left-radius: 4rem;
                  border-bottom-right-radius: 4rem;
                  border-radius: 2rem 4rem;
                  border-radius: 2rem 4rem 0;
                  border-radius: 2rem 4rem 0 1rem;
                }

Codigo del eliptico

              
                border-radius-2 {
                  border-radius: 2rem / 4rem;
                  border-radius: 2rem 4rem / 4rem 8rem;
                }

Diferencias entre clip y origin

Difencias

  1. background-clip su imagen enpieza desde el borde y si lo disminuyes recortara algunos pedasos

  2. background-origin si su imagen enpieza desde el contenido y no cabe toda la imagen se empezara a desbordar del contenido

  3. background-clip afecta a todo el fondo (la imagen y el color de fondo)

  4. background-origin afecta solo a la imagen

Background Attachment

background-attachment define cómo se comportará la imagen de fondo al desplazarse por la pagina, con esto podemos lograr efectos interesantes.

Valores

  1. scroll

    la imagen de fondo se esta llendo junto al contenedor cuando scrolleamos la pantalla

  2. fixed

    Con este valor vamos a lograr un efecto de paralax, mientras que movemos la caja se visualizara las partes de la imagen

codigo

              
                .bg-attachment {
                  background-color: skyblue;
                  background-image: url(../assent/puesta-sol.jpg);
                  background-repeat: no-repeat;
                  background-size: cover;
                  width: 100%;
                  height: 50vh;
                  background-attachment: scroll;
                  background-attachment: fixed;
                }

Ejemplo

Shorthand Background
