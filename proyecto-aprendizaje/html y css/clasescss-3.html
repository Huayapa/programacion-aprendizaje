Sombras a las cajas (box-shadow)

Podemos aplicar sombras a nuestras cajas o a los textos y existen 3 tipos de sombra

  1. Sombra a la caja

    Para esto hacemos uso de box-shadow y estos son los valores que le puedes otorgar:

    1. movimiento-x
    2. movimiento-y
    3. blur-radius
    4. spred-blur
    5. color
    6. inset (crea una sombra interna)

    Codigo

                          
                        .box-shadow {
                          box-shadow: 1rem 1rem 1rem 1rem #0d2e69;
                          box-shadow: -1rem -1rem 1rem 1rem #0d2e69;
                          box-shadow: 1rem 1rem 1rem 1rem #0d2e69 inset;
                        }

    Ejemplo

  2. Sombra al texto

    Para poder aplicar sombras a tus textos debes usar la propiedad text-shadow y estos son sus valores:

    1. movimiento-x
    2. movimiento-y
    3. blur-radius
    4. color

    Codigo

                          
                        .text-shadow {
                          font-size: 2rem;
                          text-align: center;
                          text-shadow: -1rem -1rem .4rem #0d2e69;
                        }

    Ejemplo

    soy un texto

  3. Sombras Multiples

    Nos permite crear varias sombras en la misma propiedad, solo debes poner otra ,.

    Codigo

                          
                        .text-shadow {
                          text-shadow: 1rem 1rem 1rem 1rem #0d2e69,
                          1rem 1rem 1rem 1rem #055 inset;
                        }

    Ejemplo

  4. Filtro (drop-shadow)

    Estos son filtros que nos añadiran mas variedad a nuestras sombras, cada de estos filtros son funciones y mostraremos uno de ellos ya que son varios.

    Funcion

    • drop-shadow

      Es una funcion que dara sombra a las imagenes con transparencia y puede recibir estos valores:

      1. movimiento-x
      2. movimiento-y
      3. blur-radius
      4. color

      Codigo

                                  
                              .filter-shadow img {
                                filter: drop-shadow(1rem 1rem 1rem red )
                              }

      Ejemplo

Degradados

Los degradados la logica que debemos entender antes de aplicarlos y hay estas 3 propiedades:

Para facilitar el uso de los degradados podemos usar cssgradient o colorzilla ya que tiene una interfaz entendible a la vista pero primero debes entender como se usas las gradiantes.

Recuerda que

  • Estas son funciones dentro del background

  • Lo recomendable es usar el background-image porque es la propiedad especifica

  • Puede aceptar cualquier tipo de colores(rgb,hsl,nombres,codigo hexadecimal)

  • En estas funciones al poner 0 grados debes llevar siempre deg

  1. lineales (linear-gradient)

    Puedes agregar cualquier cantidad de colores , el grado del color tambien puedes poner cuanto % tendra cada color y se aplicaran por linea

    codigo

                          
                        .linear-gradient {
                          background-image: linear-gradient(red, green );
                          background-image: linear-gradient(red, green, blue, orange );
                          background-image: linear-gradient(0deg, red, green, blue, orange );
                          background-image: linear-gradient(45deg, red, green, blue, orange );
                          background-image: linear-gradient(45deg, red 20%, green 10%, blue 10%, orange 90% );
                          background-image: linear-gradient(90deg, red 30%, white 30% 70%, red 30% );
                        }

    Ejemplo

  2. Radiales (radial-gradient)

    radial-gradient por defecto es circular y acepta los mismo valores que el lineal pero tambien puede ajustar los gradiados con circle 2rem

    Una diferencia es que en radial-gradient no puedes usar los grados para moverlos, debes usar direcciones at left top

    codigo

                          
                        .radial-gradient {
                          background-image: radial-gradient(circle 2rem cyan, magenta, yellow);
                          background-image: radial-gradient(circle 4rem, cyan 30%, magenta 30% 80%, yellow);
                          background-image: radial-gradient(circle 100px, cyan 30%, magenta 60%, yellow 90%);
                          background-image: radial-gradient(circle 100px 50px, cyan 30%, magenta 60%, yellow 90%);
                          background-image: radial-gradient(ellipse 100px at left top, cyan 30%, magenta 60%, yellow 90%);
                        }

    Ejemplo

  3. conicos (conic-gradient)

    Lo que hace es generar una proyecion conica, (como es casi reciente no estaran las sugerencias de emmet) y acepta casi los mismos valores que los anteriores funciones

    codigo

                          
                        .conic-gradient {
                          background-image: conic-gradient(red, green);
                          background-image: conic-gradient(red 50%, green 60%, blue 80%);
                          background-image: conic-gradient(from 90deg, red 0deg 90deg, green 120deg 240deg, blue 270deg);
                          background-image: conic-gradient(from 180deg, red 0deg 90deg, green 120deg 240deg, blue 270deg);
                          background-image: conic-gradient(from 90deg, red 0deg 120deg, green 120deg 240deg, blue 240deg);
                        }

    Ejemplo

Patrones con Degradados

Existen 3 funciones auxiliares que nos permiten generar patrones a la hora de implemetar los degradados para formar patrones visuales

Funciones

Recueda que estas funciones deben estar dentro de background-image y son las mismas funciones que vimos anteriormente solo que le agregamos repeating.

  1. repeating-linear-gradient

    Repetira las lineas si el valor que le distes no llena la caja

    Codigo

                        
                      repeat-gradient-lineal {
                        background-image: repeating-linear-gradient(red 0 10px, green 10px 20px, blue 20px 30px);
                      }

    Ejemplo

  2. repeating-radius-gradient

    Repetira las lineas si el valor que le distes no llena la caja

    Codigo

                        
                      repeat-gradient-radial {
                        background-image: repeating-radial-gradient(circle 4rem, cyan 0 10px, magenta 10px 20px, yellow 20px 30px);
                      }

    Ejemplo

  3. repeating-gradient-conic

    Repetira las lineas si el valor que le distes no llena la caja

    Codigo

                        
                      repeat-gradient-conic {
                        background-image: repeating-conic-gradient(red 0 8%, yellow 8% 16%, black 16% 24%);
                      }

    Ejemplo

filtro(filter)

Los filtros se pueden usar en sofware de diseño o photoshop o filtros de instagram, tenemos muchos filtros que podemos aplicar a nuestros elementos.

Funciones

los filtros son mas notorios si se trabaja en una imagen.

  1. blur()

    Aplica un filtro de desenfoque

    Codigo

                          
                        .card-blur  img {
                          filter: blur( .17rem );
                        }

    Ejemplo

  2. brightness()

    Aplica la intencidad de la luz y acepta 1 o 0 (eso quiere decir que usaras decimales) ya que si pones mas de 1 el brillo sera muy alto

    Codigo

                          
                        .card-brightness img {
                          filter: brightness( .7 );
                        }

    Ejemplo

  3. contrast()

    Aplica el contraste al elemento y vamos de 0 a 1 (se puede poner mas de 1 pero se satura la imagen)

    Codigo

                          
                        .card-contrast img {
                          filter: contrast( .5 );
                        }

    Ejemplo

  4. grayscale()

    Como su nombre lo dice es que aplicara la escala de los grises y ponemos de 0 a 1.

    Codigo

                          
                        .card-grayscale img {
                          filter: grayscale( 1 );
                        }

    Ejemplo

  5. sepia()

    Es lo mismo que la funcion grayscale pero este trabaja en los colores mas amarillos y cafes

    Codigo

                          
                        .card-sepia img {
                          filter: sepia( 1 );
                        }

    Ejemplo

  6. hue-rotate()

    Aplica el color dependiendo del grado que pongas (recuerda los grados por colores)

    Codigo

                          
                        .card-hue-rotate img {
                          filter: hue-rotate( 90deg );
                        }

    Ejemplo

  7. invert()

    Invierte los colores y estos sirven con los valores 0 y 1 (no sirven con decimales)

    Codigo

                          
                        .card-invert img {
                          filter: invert( 1 );
                        }

    Ejemplo

  8. opacity()

    Aplica opacidad a tu elemento y acepta de 0 a 1

    Codigo

                          
                        .card-opacity img {
                          filter: opacity( 0.5 );
                        }

    Ejemplo

  9. saturate()

    Aplica saturacion al elemento y va de 0 a 1 ya que usar mas satura la imagen demasiado

    Codigo

                          
                        .card-saturate img {
                          filter: saturate( 2 );
                        }

    Ejemplo

Filtros Multiples

Los filtros multiples son para poder aplicar varios filtros a un solo elemento y se aplica separandolos por espacio cada funcion, en la misma propiedad.

Codigo

    
      .filtro-multiple {
        filter: blur(.1rem) hue-rotate(270deg) opacity(.75) invert(1);
      }

Filtros a Fondos (backdrop-filter)

Podemos aplicar filtros a los fondos a los elementos y se aplica usando la propiedad backdrop-filter y puedes poner la funciones de los filtros.

Tambien puedes agregar filtros multiples al igual que los anteriores

Codigo

  
    .relative {
      position: relative;
    }
  
  
  
    .backdrop-filter {
      position: absolute;
      top: 0;
      left: 0;
      width: 100%;
      height: 100%;
      display: flex;
      justify-content: center;
      align-items: center;
      text-align: center;
    }
  
  
  
    .backdrop-filter h3 {
      border-radius: .5rem;
      padding: 1rem;
      font-size: 1.5rem;
      color: #fff;
      background-color: rgba(255, 102, 255, 0.5);
      backdrop-filter: blur(.5rem);
    }

Mascaras (clip-path)

Las mascaras son efectos para imagenes o elementos que le quieras agregar interes sin necesidad de usar una aplicacion para graficos

clip-path

clip-path es la propiedad que debemos usar y sirve para aplicar el efecto de enmascaramiento.

Esta propiedad va a ser uso de 4 funciones, uno de circulo circle(), otro de elipse ellipse(), otra de manera rectagular y otro de poligonos polygon().

  • Circle()

    Esta funcion pueden aceptar unidades de medida para especificar el tamaño del radio y tambien puedes especificar la direccion con at y la direccion at left bottomo at 30% 70%.

    Codigo

        
      clip-path-circle {
        clip-path: circle();
        clip-path: circle(200px);
        clip-path: circle(100px at left top);
        clip-path: circle(100px at 23% 50%);
      }

    Ejemplo

  • ellipse()

    Esta funcion debe aceptar 2 unidades de medidad uno para el y y el otro para el x y tambien puedes especificar la direccion con at y la direccion at left bottomo at 30% 70%.

    Codigo

        
      clip-path-ellipse {
        clip-path: ellipse();
        clip-path: ellipse(200px 100px);
        clip-path: ellipse(100px at left top);
        clip-path: ellipse(100px at 23% 50%);
      }

    Ejemplo

  • inset()

    Debe tener siempre algun valor ya que no funcionara, sirve como si fuera un padding hacia su contenido y puede aceptar hasta 4 unidades de medida, tambien puedes añadir el valor round (redondeara los costados).

    Codigo

        
      clip-path-inset {
        clip-path: inset(1rem);
        clip-path: inset(1rem 2rem);
        clip-path: inset(1rem 2rem 3rem);
        clip-path: inset(1rem 2rem 3rem 4rem);
        clip-path: inset(1rem round 1rem);
      }

    Ejemplo

  • polygon

    Esta funcion necesita valores para que funcione, sirve para poder crear figuras geometricas ya que puedes poner que cada punto este en la direccion que quieras(puede ser nuy confuso)

    Esta funcion puede ser algo confusa y tediosa al crear formas geometricas es por eso que hay una pagina que te ayudara se llama: clip path generator.

    Codigo

        
      clip-path-polygon {
        clip-path: polygon(0 0, 100% 0, 50% 100%);
        clip-path: polygon(0% 0%, 0% 100%, 25% 100%, 25% 25%, 75% 25%, 75% 75%, 25% 75%, 25% 100%, 100% 100%, 100% 0%);
      }

    Ejemplo

Sitio One Page Scroll

Aqui veremos un par de efectos que seran sobre el desplazamiento de la pagina (scroll).

Propiedades

  1. scroll-margin

    Tiene 4 propiedades juntas (top, right, left, bottom) y sirve para el margen del scroll.

    Ejemplo de Problema

    Cuando tenemos algun contenido con position: sticky; y queremos ir a alguna seccion con tu enlace, habra un problema ya que la seccion empieza desde el top 0 y el contenido con una position: sticky; se comera esa parte

    Soluciones

    • Podemos usar padding-top para solucionarlo pero no es la mejor forma.

    • La mejor forma de solucionarlos es usando el scroll-margin-top y usar la unidad de medida ex ya que esta trabaja con la altura y un ex es igual a la mitad de 1rem

    Codigo

        
      [id] {
        scroll-margin-top: 7,5ex;
      }

  2. scroll-behavior

    Sirve para poner animaciones cuando realizemos algun tipo de scroll (En este ejemplo lo usamos para los enlaces)

    Codigo

          
        html {
          scroll-behavior: smooth;
        }

Ajustes del Scroll Efectos Slides

Sirve Cuando queremos mover el scroll suavemente y automaticamente centrar el contenido para que se vea como una presentacion de diapositivas PowerPoint

Importante

  • Para poder que pueda funcionar es inportante que el heighteste al 100vh y despues ponerle overflow-y: scroll ya que sin esto no funcionara.

  • las cajas hijas deben tener el scroll-snap-align y los padres scroll-snap-type para que funcione

Propiedades:

  1. scroll-snap-type

    Sirve para indicar la direccion y el efecto que hara, acepta 2 valores y cada valor tiene distintos valores: 

          
        scroll-snap-type:
         [x | y | inline | block |both] }Estos valores puede aceptar
                                    el primero y es para la direccion
         [madatory | proximity] }Esto valores son para el segundo y 
                                  sirven para los efectos
        ;

  2. scroll-snap-align

    Sirve para cuando realizemos un scroll se centre bien y acepta (none, start, end, center)

Codigos

    
      .slides {
        width: 100%;
        height: 100vh;
        overflow-y: scroll;
        scroll-snap-type: block mandatory; }Propiedad 1
      }
    
  
    
      .slide {
        width: 100%;
        height: inherit;
        background-color: #1e2345;
        scroll-snap-align: center; }Propiedad 2
      }
    
  
    
      .slide:nth-child(even) {
        background-color: #108eb4;
      }
    
  
    
      .slide-container {
        width: 80%;
        height: inherit;
        margin: 0 auto;
        display: flex;
        flex-direction: column;
        justify-content: center;
        align-items: center;
        font-size: 3vw;
        color: #d9e8f0;
      }

Texto con Degradados

Ahora vamos aprender a usar textos con degradados (no los explicamos antes en los degradados porque puede servir cuando usemos diapositivas)

Importante

  • Las propiedades para hacer textos con degradados aun no estan soportados es por eso que vamos a usar los prefijos del navegador

  • Debe tener un background-clip: text; y sirve para el color de fondo y la imagen comienze desde el texto

  • Hay un poco de latencia al aplicar gradiente al texto, es por eso que no debes abusar mucho.

text-fill-color

Debe usarse los prefijos, Sirve para que lo que no sea texto se convierta en transparent text-fill-color: transparent; es como ignorar la propiedad color.

Codigo

  
    .gradient-text {
      background-image: linear-gradient( 45deg, magenta, yellow);
      -ms-background-clip: text;
      -moz-background-clip: text;
      -webkit-background-clip: text;
      background-clip: text;
      -ms-text-fill-color: transparent;
      -moz-text-fill-color: transparent;
      -webkit-text-fill-color: transparent;
    }

Transiciones

Hay varias propiedades para las transiciones:

Propiedades

transition es un shorthand de estas propiedades

  1. transition-property

    Sirve para indicar a que propiedad le va aplicar una transicion

  2. transition-duration

    Es la duracion que tendra la transicion (acepta segundos o milisegundos)

  3. transition-timing-function

    Es para aplicar el efecto de la transicion, puede aceptar estas funciones none | linear | ease | ease-in | ease-out | ease-in-out | steps

  4. transition-delay

    Es el tiempo de espera antes de que comienze la transicion

Inportante

si usas el shorthand, las antiguas propiedades ya no funcionaran por la hoja en cascada, hay una solucion pero es de mala practica no obstante puedes usarlo si tu pagina usa pocas animaciones y solo tienes que poner all en ves de definir una propiedad en transition

Codigo

  
    .transitions .box  {
      background-color: magenta;
      transition-property: background-color;
      transition-duration: 500ms;
      transition-timing-function: ease;
      transition-delay: 0.5s;
      transition: border-color 2s linear 0.3s;
      transition: all 2s ease-in-out 250ms;
    }
  

  
    .transitions .box:hover {
      background-color: cyan;
      border-color: red;
      border-radius: 2rem;
    }

Porque no usar el all en las transiciones

El valor que le das al all afecta a todas las propiedades incluyendo a la clases que le das anteriormente, es por eso que se considera mala practica si no sabes controlar al 100% las propiedades que vas a animar.

El algoritmo de Css y su efecto sobre las transiciones

Estas transiciones son afectadas con el algoritmo de CSS (cascada, especificidad y por la herencia) es por eso que al recargar la pagina suceden las animaciones por un instante

Transformaciones en 2D

Existen 2 tipos de transformaciones en 2d y 3d mostraremos primero lo 2d

Sirven para crear transformaciones y su propiedad es transformExisten 4 transformaciones en 2d que son las mas conocidad y mas usadas:

Inportante

  • En este ejemplo vamos a transformar la imagen y no los border para que se pueda apreciar mejor.

  • Para que funcione debemos añadir la propiedad transition y el valor que animaremos con su respectivo segundo y forma.

  • Cuado uses la propiedad transform debes añadir sus funciones respectivas.

  1. Traslacion en x o y

    Sirve para trasladar el elemento su funcion se llama translate(debes añadir 2 valores para esto), si quieres moverlo solo un lado puedes usar translateX o translateY, puedes utilizar cualquier unidad de medida

    Codigo

      
    .transform-2d-translate img {
      translation: transform 2s ease-in-out;
    }
  

      
    .transform-2d-translate img:hover {
      transform: none;
      transform: translateX(4rem);
      transform: translateY(4rem);
      transform: translateY(-4rem);
      transform: translate(-4rem, -4rem);
      transform: translate(50%, 50%);
    }

    Ejemplo

  2. Escala (aumento de tamaño) en x o y

    Sirve para aumentar o reducir el tamaño del elemento, debes usar la funcion scale() (debe tener 2 unidades de medida) y si solo quieres aplicar a una direccion debes usar scaleX() o scaleY().

    Codigo

      
    .transform-2d-scale img {
      translation: transform 2s ease-in-out;
    }
  

      
    .transform-2d-scale img:hover {
      transform: scaleX(2);
      transform: scaleY(.5);
      transform: scale(2, .5);
      transform: scale(-.5, -.5);
    }

    Ejemplo

  3. Rotacion

    Sirve para poder rotar al elemento, debes usar la funcion rotate() (acepta valor de rotacion 240deg) y tambien tiene la funcion rotateX() o rotateY() O rotateZ()(para apreciar mejor el x o y debemos verlo en 3d)

    Codigo

      
    .transform-2d-rotate img {
      translation: transform 2s ease-in-out;
    }
  

      
    .transform-2d-rotate img:hover {
      transform: rotateX(90deg);
      transform: rotateY(90deg);
      transform: rotateZ(90deg);
      transform: rotate(90deg);
    }

    Ejemplo

  4. skew (sesgar) en x o y

    Sirve para crear sesgados y solo existen en 2d, debemos usar la funcion skew() (acepta valor de rotacion 240deg) y tambien las funciones skewX() o skewY()

    Codigo

      
    .transform-2d-skew img {
      translation: transform 2s ease-in-out;
    }
  

      
    .transform-2d-skew img:hover {
      transform: skewX(20deg);
      transform: skewX(-20deg);
      transform: skewY(20deg);
      transform: skewY(-20deg);
      transform: skew(20deg, 20deg);
    }

Animaciones (conceptos basicos)

Ahora que terminamos con las transformaciones, toca hablar sobre las animaciones

Las animaciones son para crear movimientos con efectos a tu gusto que podemos aplicar en nuestra pagina web, tienes algunas propiedades del transition pero este shorthand del las animaciones tiene mas propiedades.

animation

Animacion es un shorthand de estas 8 propiedades en este mismo orden:

  1. animation-name

  2. animation-duration

  3. animation-timing-function

  4. animation-delay

  5. animation-iteration-count

  6. animation-direction

  7. animation-fill-mode

  8. animation-play-state (no es necesario)

De todas esas propiedades solo es obligatorio poner los primeros 2, Las propiedades agregadas para las animaciones son estas:

  • animation-iteration-count

    Cuantas veces se va agregar la iteracion (un bucle o repeticion).

  • animation-direction

    La direccion de la animación para saber donde se enpieza

  • animation-fill-mode

    Especifica el modo en que una animacion aplica sus estilos

  • animation-play-state

    Sirve para el estado de la animación, mayormente no se usar ya que para pausar una animacion se usa javascript.

Forma de agregar la animacion basica:

  1. El @keyframe es como si fuera la linea del tiempo de la animacion.

  2. el "myAwesomeAnimation" es el nombre que le das a esta funcion (puedes poner el nombre que quieras) y sirve para ponerlo en la propiedad animation-name.

  3. para saber cual es el inicio debes agregar el from (todo lo que pongas en el from enpecera al iniciarse) y para saber cual es el final debes poner el to (la animacion terminara dependiendo que le pongas). 


  @keyframe myAwesomeAnimation {
    from {
      opacity: ;
      transform: translateX(100%)
    }

    to {
      opacity: ;
      transform: translateX(0)
    }
  }



  .my-animation {
    animation-name: myAwesomeAnimation;
    animation-duration: 3s;
  }

Linea de tiempo y fotogramas claves en CSS

Como puedo agregar mas de un fotograma a mi animacion aparte de usar el from (Fotograma Inicio) y to (Fotograma Final) e

Esto son la linea de tiempo con sus fotogramas en una aplicacion de animacion, las lineas son el tiempo de duracion y los puntos son los cambios de la animacion

En css la linea de tiempo funciona por porcentajes (%), eso quiere decir que en ves de usar from o to debesmo usar 0% y 100% para que los demas porcentajes (1% hasta 99%) sirvan como los fotogramas intermedios, recuerda que a mayor tiempo de duracion mas lento es la animacion

Codigo


  @keyframe myAwesomeAnimation2 {
    0% {
      opacity: 0;
      transform: translateX(0);
    }

    50% {
      opacity: 0.5;
      transform: translateX(-50%);
    }

    75% {
      transform: translateX(-0%);
    }

    100% {
      opacity: 1;
      transform: translateX(100%);
    }
  }



  .my-animation-2 {
    animation: myAwesomeAnimation2 2s;
    animation-iteration-count: infinite;
  }

Ventana Modal

Ahora vamos a aprender a hacer una ventana modal sin la necesidad de Javascript

Recomendacion

A los objetos interactivos como un boton, es recomendable ponerle un tamaño entre 2.5rem o 3 rem ya que esto es un aproximado al area de un dedo usando un celular.

pointer-events

hace que los elementos (enlaces) no sean interactivos.

Codigo

  
    .modal {
      position: fixed;
      top: 0;
      right: 0;
      bottom: 0;
      left: 0;
      z-index: 997;
      display: flex;
      justify-content: center;
      align-items: center;
      background-color: #0008;
      opacity: 0;
      pointer-events: none;
      transition: opacity 500ms ease-in-out;
    }
  

  
    .modal-container {
      position: relative;
      border: thin double #d00;
      border-radius: 1rem;
      padding: 2rem;
      width: 70%;
      height: 70vh;
      display: flex;
      flex-direction: column;
      text-align: left;
      overflow: hidden;
      background-color: #fff;
    }
  

  
    .modal:target {
      opacity: opacity;
      pointer-events: auto;
    }
  

  
    .modal-close {
      position: absolute;
      top: 1rem;
      right: 1rem;
      width: 3rem;
      height: 3rem;
      font-size: 2rem;
      font-weight: bold;
      text-align: center;
      text-decoration: none;
      line-height: 3rem;
      color: #fff;
      background-color: #d00;
      border-radius: 50%;
      transition: background-color 300ms linear,
         transform 300ms ease-in-out;
    }
  

  
    .modal-close:hover {
      background-color: #a00;
      transform: scale(1.2);
    }
