Etiquetas semanticas estructurales

es una etiqueta contenedora que no tiene semantica
header - cabecera de un sitio web o de una seccion
main - Define la seccion principal del documento, solo puede existir una etiqueta main por documento
footer - Pie de pagina de un sitio web o de una seccion
nav - representa una navegacion
article - representa una seccion autocontenido (que por si sola se explica)
aside - representa contenido complementario o secundario(puede ser para anuncio)
section - representa una seccion generica
address - representa una direccion fisica o de contacto

Elementos de Formularios

radio (solo se puede marcar una alternativa)

Estos son input semanticos

parecen iguales al input text pero estos añaden una ayuda o funciones a dispositivos mobiles

Atributos de Imputs y Formularios

Son elementos que puedes inplementar ala etiqueta input y formularios, estos son algunos de esto:

atributos para el input text

  • readonly - no se puede cambiar lo que pusistes (puede servir para usuarios)

  • disable - no se puede interactuar con el input

  • placeholder - escribe unos textos en donde puedes escibir (sirve para indicar)

Existen varios tipos de metodos para el form:

  • method="GET" sirve para ponerlo al URL
  • method="POST" envia de forma oculta
  • autocomplete="off" para que no salgan las sugerencias de haber buscado algo antes
  • <label> sirve para poner el texto visible al lado, el atributo for del label es para vincular con el id de un input
  • required no dejara enviar hasta que este completo
  • pattern="" sirve para poner una expresion regular (si quieres que salga un mensaje de regulacion personalizada usar el atributo title="" )

Metaetiquetas para SEO y Moviles

pueden ser de buena ayuda para el posicionamiento de busqueda(contribuir al CEO de tus documentos HTML)


etiquetas principales:

Hay 2 etiquetas que ya sabemos que son title y meta description hay una tercera que nos puede servir.

Esto es para crear tu URL de forma canonica (que es la URL principal de tu dominio).

<link rel="canonical" href="http://midominio.com/url-canonica">

Estas son etiquetas que complementan a SEO:

  • favicon = se le llama asi porque es para poner un icono en tu pestaña de navegador web.
    <link rel="icon" href="img/imagen.png">
    (ojo recomendable usar en formato png).

  • appleTouchIcon = se le llama asi ya que apple lo propuso para sus moviles, sirve para poner icono a los moviles(casi igual que favicon)
    <link rel="apple-touch-icon" href="img/hmtl.png">
    de igual manera usar formato png

  • themeColor = sirve para que la barra este de color en los moviles
    <meta name="theme-color" content="#FF6600">
    (el color debe estar en codigo hexadecimal)

Meta etiquetas para Redes Sociales

sirve para cuando queremos compartir nuestra url a redes sociales

para hacer esto debemos usar el open graf (la mayoria de redes las usan, puedes copiar y pegar lo que te sale en esa web solo modificar algunas cosas)


los mas inportantes son:

<meta property="og:title" content="Clases Html y Css" />
<meta property="og:description" content="En esta pagina solo es para practicar html" />
<meta property="og:image" content="https://tudominio.com/img/html.png" />
<meta property="og:url" content="http://midominio.com/url-canonica" />

los de twitter (ya que ellos poseen algo diferente)

<meta name="twitter:cart" content="summary">
<meta name="twitter:site" content="@usuario">
