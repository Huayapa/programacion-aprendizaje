Flexbox CSS

Introduccion

Flexbox (Modulo de caja Flexible) es un metodo en el que nos ayuda a distribuir el espacio de los items de una interfaz y mejora las capacidades de la alineacion.

Conceptos Basicos

Lo primero que debemos saber y entender de Flexbox es su terminologia, ya que realmente Flexbox es un sistema unidimencional (puedes tener filas y columnas pero no ambas)

Para poder usar el flexbox debemos usar el atributo display:___;

Factores de Crecimiento (flex-grow)

Flex-grow

Cuando la caja flexbox tenga algun espacio disponible o sobrante los hijos tomaran todo ese espacio gracias a flex-grow y NO puede aceptar numero negativos

Como invocar a la propiedad?

Este es el valor por defecto


            .flex-grow {
              flex-grow: 0;
            }

Inportante!

Esta propiedad debe vincularse a sus hijos NO el padre

Ejemplos

Para que funcione la propiedad debe estar en uno por lo menos, estando en 1 los item hijos tomaran el espacio sobrante igualmente. 


          .flex-grow {
              flex-grow: 1;
          }

pero cuando uno de ellos tiene el flex-grow en 2 o mas podra agarrar mas espacio sobrante (en este caso el 2 contenido tendra 3) 


          .flex-grow {
              flex-grow: 1;
          }
        

          .flex-grow:nth-child(2) {
              flex-grow: 3;
          }

Factor de Reduccion (flex-shrink)

flex-shrink

Sirve para cuando la caja flexbox NO tenga espacio sobrante, me refiero a que si escroleas hasta dejar un espacio pequeño los hijos se reduciran , NO se aceptan valores negativos.

Como invocar la propiedad?

Este es el valor por defecto


          .flex-shirink {
              flex-shrink: 1;
          }

Ejemplos

Esto es casi igual que flex-grow pero en al revez, Si quieres ver que pasa debes reducir la ventana.

Tamaño de los hijos flexbox (flex-basis)

flex-basis

Es el tamaño del elemento hijo dentro de la línea de la caja flexbox y al momento que exista esta propiedad ignorara el width o height (dependiendo si esta en fila o columna).

Inportante

Si la caja flexbox tiene direccion de fila, flex-basis representara el width.

Si la caja flexbox tiene direccion de columna, flex-basis representara el height.

El valor por defecto es auto

Como se invoca la propiedad?

Este es el valor por defecto. 


          .flex-basis {
                flex-basis: auto;
          }

Ejemplos

Aqui lo ponemos con un valor de 150px, que en este ejemplo es el remplazo de width. 


          .flex-basis {
              flex-basis: 150px;
          }

shorthand (flex)

Al igual que los elementos padres, existe un shorthand para los elementos hijos que siplifica estas tres propiedades (flex-grow, flex-shrink y flex-basis) a uno solo.

Como se invoca la propiedad?


          .flex {
              flex: 0 1 auto;
          }
esta es la propiedad shorthand para los elementos hijos (psdt: esta en el valor por defecto)

Orden y Alineación de hijos flexbox (orden y align-self)

Nosotros podemos controlar el orden de los elementos y este valor debe estar hacia los hijos para que funcione.

Propiedad: orden

Representa el orden que tendrán los elementos hijos en la caja flexbox.

  • Valor por defecto es 0
  • Se pueden aceptar valores negativos y positivos
  • Un valor menor siempre irá antes que un valor mayor

Ejemplos

Como todos los valores enpiezan con 0 al poner un numero negativo a cualquiera de los valores tomara el primer lugar y si es un numero mayor que 0 sera el ultimo. 


            .flex-orden:nth-child(2) {
                orden: -1;
            }
          

            .flex-orden:nth-child(4) {
                orden: 2;
            }

Propiedad: align-self

Sobreescribe el valor de la propiedad align-items soló para el elemento hijo especificado, valores que usa (ya estos valores ya se saben sus usos):

  • stretch
  • flex-start
  • flex-end
  • center
  • baseline

Ejemplo
